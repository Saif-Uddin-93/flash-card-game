Flashcard Instructions

Zoom in on an image by clicking/tapping on it.

Main menu
  • 1. Settings - Access game preferences and customize settings.
  • 2. Trophy - Displays user's highest score.
  • 3. Exit - Allows users to exit the game and return to the main menu.
  • 4. Progress Bar - Represents user progress through the game
  • 5. Categories - Select a category to guess words from multiple images.
  • 6. Notifications - Displays short information about different parts of the layout. Can be disabled in the Settings area.
Gameplay
  • 7. Hint Link - Click to flip the flashcard and reveal a short description for the image.
  • 8. Hearts - Represents the number of attempts remaining.
  • 9. Score - Shows the current score for the ongoing game.
  • 10. Image Index - Indicates the current image out of the total in the current category.
  • 11. Underscores - Displays underscores representing the word to be guessed.
  • 12. Mobile - Only visible for tablets/mobiles. Tap to trigger the off-screen keyboard
Reset the Game

If you wish to start the game anew:

  1. Go to the settings ⚙️ in the game.
  2. Find the reset option under Local Storage.
  3. Click on the Reset option to reset your game progress.
